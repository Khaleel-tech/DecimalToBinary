# Decimal to Binary Converter 🔢➡️💻

This repository contains implementations of converting a **Decimal number to Binary** in three popular programming languages:

- 🐍 Python
- 🧠 C++
- 🔧 C

---

## 📁 Files

| File Name       | Language |
|----------------|----------|
| `decToBin.py`  | Python   |
| `decToBin.cpp` | C++      |
| `decToBin.c`   | C        |

---

## 🚀 How to Run

### 🐍 Python
```bash
python DecToBin.py
```
### 🔧 C
```bash
gcc DecToBin.c -o dedToBin
./DecToBin
```
### 🧠 C++
```bash
g++ DecToBin.cpp -o DecToBin
./DecToBin
```
